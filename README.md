# 🛒 Superstore Sales EDA Case Study  
## 📖  Project Overview  
This project focuses on performing Exploratory Data Analysis (EDA) on the popular Superstore Sales dataset using Python and Jupyter Notebook.  
The dataset contains multiple data quality issues such as:  
•	Missing values  
•	Inconsistent formats  
•	Duplicate records  
•	Incorrect categorical values  
•	Data type mismatches  
The objective of this project is to clean, transform, and analyze the dataset to uncover meaningful business insights while ensuring the notebook remains reusable for future datasets with similar structures.  

## 🎯 Objectives  
•	Perform systematic data cleaning and preprocessing  
•	Handle missing and inconsistent data  
•	Track impacted rows and distinct Order IDs during transformations  
•	Conduct exploratory data analysis (EDA)  
•	Generate meaningful business insights through visualizations  
•	Build a reusable and well-documented Jupyter Notebook workflow
